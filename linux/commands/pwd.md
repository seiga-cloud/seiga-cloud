# pwd command

## Overview
`pwd` (Print Working Directory) displays the full path of the current directory.

## Syntax
pwd

## Example

$ pwd
/home/user/linux-study
This shows the current directory path.

## Options
| Option | Description |
| -L | show logical path |
| -P | show physical path |

Example:
$ pwd -P

## Use Case

-Confirm current directory
-useful in scripts
-Avoid path mistakes when operating files

## Summary

`pwd` is a basic Linux command used to display the current working directory.
